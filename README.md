静的ファイル（HTML/CSS/JS）をベースに、WordPressの独自テーマとして動的に組み上げたプロジェクトです。

## 🛠 使用技術
- **CMS**: WordPress
- **言語**: PHP, HTML5, CSS3, JavaScript

## 💡 実装の事実と特徴
- **テンプレート階層の適用**: rchive.php, single.php, page-{slug}.php など、WordPressのテンプレート階層の規約に従って適切にファイルを分割・構成しています。
- **カスタムフィールドの活用**: 固定ページ（会社概要など）において、クライアントが管理画面からHTMLを触らずにテキストや画像を更新できるよう、カスタムフィールドを組み込んで動的に出力しています。
- **動的ループ処理の実装**: 記事一覧や最新の投稿などを取得するため、WordPressループ（WP_Query や have_posts() 等）を用いてデータをフロントエンドに展開しています。
- **静的デザインの完全再現**: 元となる静的サイトのデザインやJavaScriptの挙動（パンくずリストやメニューなど）を一切崩すことなく、動的なCMSへと昇華させました。
