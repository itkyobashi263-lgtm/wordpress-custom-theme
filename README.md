# WordPress オリジナルテーマ開発

静的ファイル（HTML/CSS/JS）をWordPress化し、オリジナルのテーマとして構築したプロジェクトです。

## 🛠 使用技術
- **WordPress**
- **PHP**
- **HTML/CSS/JavaScript**

## 💡 制作のポイント
- テンプレート階層（page-company.php、rchive.php、single.phpなど）を正しく理解し、規約に沿ったファイル構成で作成しました。
- カスタムフィールドを活用し、クライアントが管理画面から運用しやすいサイト設計を心掛けました。
- 静的サイトのデザインを崩すことなく、動的化（ループ処理、パンくずリストなど）を実装しました。
